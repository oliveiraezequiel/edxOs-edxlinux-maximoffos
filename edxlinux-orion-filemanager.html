Como instalar o Orion File Manager

Siga os passos abaixo para baixar e descompactar o pacote diretamente do repositório oficial:

  1. Baixar o arquivo (usando a URL RAW correta do GitHub)
    2. wget https://github.com/oliveiraezequiel/edxOs-edxlinux-maximoffos/raw/main/orion-filemanager.tar.gz
  2. Descompactar o arquivo baixado
    3. tar -xzvf orion-filemanager.tar.gz

Depois de descompactar, entre na pasta criada e siga as instruções de instalação que estiverem lá (geralmente um README.md ou install.sh).

Comando tudo-em-um (baixa + descompacta automaticamente):

wget https://github.com/oliveiraezequiel/edxOs-edxlinux-maximoffos/raw/main/orion-filemanager.tar.gz \
  && tar -xzvf orion-filemanager.tar.gz

Dicas extras

  • Baixar direto para uma pasta específica (ex: Downloads):
    wget -P ~/Downloads https://github.com/oliveiraezequiel/edxOs-edxlinux-maximoffos/raw/main/orion-filemanager.tar.gz
  • Descompactar em uma pasta com nome específico:
    mkdir orion-filemanager && tar -xzvf orion-filemanager.tar.gz -C orion-filemanager
